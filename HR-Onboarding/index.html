These rules apply to everyone using the system â€” candidates and HR alike.

Rule 1 â€” Mobile Number is Permanent

Once a candidate enters their mobile number in Form 1 (or Walk-In Form), it cannot be changed. All future forms (Form 2, Form 3, Form 4) must use this exact same number. Changing it will break all links between the candidate's records.

Rule 2 â€” No Duplicate Mobile Numbers

Two different candidates cannot use the same mobile number. The system treats each mobile number as a unique person. If a candidate tries to register with a number already in the system, it will be blocked.

âš ï¸
Rule 3 â€” Form 2 Must Match Form 1

For online candidates, the mobile number entered in Form 2 (on interview day at the office) must be identical to the number used in Form 1. Even a single digit difference will result in the forms not being linked and the interview cannot be found.

Rule 4 â€” Walk-In Candidates Skip Form 1 and Form 2

Walk-in candidates do not fill Form 1 or Form 2. They go directly from the Walk-In Form to the interview (Form 3). The Walk-In Form plays the same role as Form 1 for them â€” it creates their initial record in the system.

âœ…
Rule 5 â€” Form 4 Works the Same for Both Paths

Whether a candidate came online or as a walk-in, Form 4 works the same way. The selected candidate comes to the office and fills it. The system links it to their profile automatically using the mobile number.

ğŸ’¡
Rule 6 â€” HR Actions Only Through the Dashboard

All HR actions â€” reviewing documents, scheduling interviews, conducting interviews, making decisions, sending messages â€” are done through the HR Dashboard. Candidates never have access to the dashboard.

ğŸ«
Rule 7 â€” QR Code is Single-Use Only

Each QR code generated for interview check-in can only be scanned once. It expires within 24 hours of generation. Attempting to scan an already-used or expired QR code will show an error. For rescheduled interviews, a new QR code is generated automatically.

ğŸ“…
Rule 8 â€” 30-Day Reapplication Rule

A candidate cannot apply again (register with the same mobile number) within 30 days of their last application. The system enforces this automatically at Form 1 submission. This prevents duplicate applications from the same candidate within a short period.

Rule 9 â€” Background Verification is Mandatory (Planned)

Once implemented, ALL selected candidates must pass background verification before an offer letter is generated. This step cannot be bypassed by HR staff â€” only an HR Manager can extend the timeline in exceptional cases. If BG fails, the application is permanently closed.

Once an offer letter is sent to a candidate, they have 7 days to accept or reject it online. If no response is received, the offer is automatically marked as expired. The HR Manager can extend the validity period if needed.
